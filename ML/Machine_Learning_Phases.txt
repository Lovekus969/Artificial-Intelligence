Machine Learning (ML) is a field of Artificial Intelligence where computers learn patterns from data and improve their performance without being explicitly programmed for every task.
Training Data
    
    Training data is the data used to teach the machine learning model.
    The algorithm studies this data and learns patterns.

Testing Data

    Testing data is used to evaluate the model after training.
    The model has never seen this data before.

Why We Split Data
    
    If we test on the same data used for training, the model may memorize instead of learning.
    This problem is called overfitting.
So we divide the dataset.

Typical split:
                      80% Training
                      20% Testing
Dataset
   │
   ├── Training Data (80%) → Train Model
   │
   └── Testing Data (20%) → Check Accuracy


Phases of Supervised Learning

In Supervised Learning, the model learns from labeled data (input + correct output).\
Example: predicting house prices with Linear Regression.

Phases:
        1. Data Collection
        Collect dataset with input and output.

2. Data Preprocessing
    Clean the data:
                Remove missing values
                Normalize data
                Remove noise

3. Train the Model
    Use training data to learn patterns.
    Algorithms:
                Linear Regression
                Decision Tree    
                Support Vector Machine
4. Testing the Model
    Use testing data to check performance.

5. Prediction
    Model predicts outputs for new unseen data.


Phases of Unsupervised Learning

    In Unsupervised Learning, the data has no labels.
    The model finds hidden patterns.
        Example: customer segmentation.

Phases
    1. Data Collection
        Only input data is available.

Phases of Reinforcement Learning
    In Reinforcement Learning, an agent learns by interacting with an environment.

    It receives rewards or penalties.
    Example: robot learning to walk or game AI.
