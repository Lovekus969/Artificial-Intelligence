Q: What is a dataset?
A dataset is a collection of data used to train or test a model.

Q: What is an instance (or Sample)?
Each row in the dataset is called an instance, sample, or data point.

Q What is a Feature?
A feature is an input variable used to predict something.

Label = the correct output value that the machine learning model tries to predict.

3.3.3 Reinforcement Learning (RL)

Reinforcement Learning is when a system learns by interacting with an environment and receiving rewards or penalties.

The model improves by trial and error.

Agent learns through interaction

    The Cycle (step by step)
    Observation → The environment sends the current state to the agent (e.g., a game screen, sensor data)
    Policy → The agent uses its policy to decide what action to take
    Action → The agent acts on the environment
    Reward → The environment evaluates the action and sends a reward signal (positive or negative)
    RL Algorithm → Uses the reward to update the policy, making better decisions next time

The Goal
The agent learns to maximize cumulative reward over time by continuously refining its policy through trial and error — without being explicitly programmed on how to act.
